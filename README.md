# Por fios de poesia
<p> This website was developed to search and share poetries.</p>
To register your poetries, it is necessary to create an account. 
This will give you access to the area for add, edit and delete your own poetry, which means that this user can only change the content inserted by itself.
All the poetries that were posted will be available on the home page.</p>
